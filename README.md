部活及びサークル活動を通じて、中学から大学生までJavaやUnityでゲーム開発をやっています。
シューティングが好みのジャンルですが、カードゲーム、謎解き、プラットフォームアクションにも手を出しており、常にコンフェスで新しいメンバーと新しいジャンルのゲーム作成に挑戦しています。
大学以降中国に留学し、その間の作品は中国語の内容となっています。
